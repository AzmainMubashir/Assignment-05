What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure and content of a web page as a tree of objects, allowing programs to interact with and manipulate the document's structure, style, and content.

2. By Class Name,
3. By Tag Name,
4. By CSS Selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that leverages event bubbling to handle events at a higher level in the DOM than the element on which the event originated. Instead of adding event listeners to multiple child elements, you add a single event listener to a parent element. This listener can then manage all events that bubble up from its child elements.

It is useful because of:

1. Efficiency: Reduces the number of event listeners, which can improve performance, especially with many child elements.

2. Dynamic Content: Simplifies handling events for dynamically added elements, as the parent listener will automatically handle them.

3. Cleaner Code: Leads to more maintainable and cleaner code by centralizing event handling.
